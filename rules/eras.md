# Eras and events

## Era and Event Rules

### Purpose

Eras and events exist only to provide rough historical context. They help users quickly understand what was happening during a person's lifetime, not to create a complete historical timeline.

### Rule 1: Use only widely recognized eras and events

Only include eras and events that are well known in general history.

Good examples:

* Renaissance
* Age of Exploration
* Industrial Revolution
* Enlightenment
* World War I
* World War II
* French Revolution

Avoid:

* Minor regional conflicts
* Short-lived political movements
* Obscure local events
* Events primarily significant within a single city, province, or small country

### Rule 2: Prefer broad eras over individual events

When both are available, prefer a major era that helps explain the historical context of the period.

Example:

* Use "Industrial Revolution" rather than several separate inventions and factory milestones.

### Rule 3: Limit event density

Do not add more than one event for any 50-year period.

If multiple major events occur within the same 50-year span, select the one that provides the most useful historical context.

### Rule 4: Choose the most globally significant event

When several events compete for the same period, prefer the event with the greatest long-term historical impact and broadest international recognition.

Example:

* Prefer "World War II" over a specific military campaign.
* Prefer "French Revolution" over a related political uprising.

### Rule 5: Avoid overlapping labels

Do not include multiple eras or events that describe essentially the same historical development.

Example:

* Use either "Renaissance" or "Italian Renaissance", not both.
* Use either "Industrial Revolution" or "Second Industrial Revolution" if only one is needed for context.

### Rule 6: Keep the timeline uncluttered

Include only the minimum number of eras and events needed to understand the period. A sparse timeline is preferred over a comprehensive one.

### Rule 7: Favor historical anchors

Select eras and events that help users immediately place a person in history.

Ask:
"Would an average educated person recognize this as a useful marker of the period?"

If not, do not include it.
