# Display Name

Use the shortest name that remains immediately recognizable to a general audience.

Never shorten a name if doing so could reasonably refer to multiple famous people.

The goal is to reduce visual clutter while ensuring users can still identify the person without ambiguity.

Guidelines:

## Prefer a widely recognized surname alone when it uniquely identifies the person.

Christopher Columbus => Columbus
Niccolò Machiavelli => Machiavelli
Leonardo da Vinci => Da Vinci
Galileo Galilei => Galileo

## Keep the full name when dropping part of it would make the identity unclear or ambiguous.

Anne Boleyn => Anne Boleyn
Mary Wollstonecraft => Mary Wollstonecraft
Catherine the Great => Catherine the Great

## Preserve established historical titles, epithets, and nicknames when they are the primary way the person is known.

Vlad the Impaler => Vlad the Impaler
Alexander the Great => Alexander the Great
Ivan the Terrible => Ivan the Terrible

## For monarchs, popes, and similar rulers, retain the regnal number and remove territorial qualifiers if unnecessary for recognition.

Richard III of England => Richard III
Louis XIV of France => Louis XIV
Pope John Paul II => John Paul II
